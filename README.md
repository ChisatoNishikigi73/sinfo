# SInfo

SInfo is a server-side Fabric mod providing an HTTP API for Minecraft server management. It facilitates remote monitoring and control through a structured interface.

## Key Features

- Event streaming via SSE (Join, Leave, Chat, Advancements).
- Performance metrics (TPS, MSPT, Memory usage).
- Remote command execution with feedback.
- Automatic vanilla item texture caching.

## Configuration

Port and authentication settings are located in `config/sinfo.json`.
